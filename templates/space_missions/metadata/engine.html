Engine Metadata

  • Name: the name of the engine, given by the manufacturer
  • Manufacturer: the code of the organisation that manufactured the engine.
  • Mass: the mass in Kg of the engine.
  • Impulse: the impulse of the engine, expressed in kN/s.
  • Thrust: the thrust of the engine, expressed in kN
  • Isp: specific impulse, expressed in seconds
  • Burn Duration: the period between the start and the end of burning process is called the burning duration, expressed in seconds.
  • Number of Chambers: number of combustion chambers.
